# Command builder

The [mc-tui](server.md#mc-tui-the-cheatsheet-as-a-tui) terminal picker as a
web page — pick a command, fill in the blanks, copy, paste. Works on phones,
nothing to install, and the item / enchantment / effect pickers are searchable
in **English or português (pt-BR)** — try `picareta` or `remendo`.

- **Playing?** Keep the `/` toggle **on** and paste into the game chat
  (press ++t++ to open it). You need to be an operator for most of these.
- **Admin on the server console or over RCON?** Turn the `/` toggle **off**.
- Browsers can't speak RCON, so commands can't run from this page — for
  pick-and-run straight to the server, use `mise run tui` in a terminal.
